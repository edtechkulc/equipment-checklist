📖 วิธีตั้งค่า Google Apps Script (ทำครั้งเดียว)
1
สร้าง Google Sheet
ไปที่ sheets.google.com → สร้าง Spreadsheet ใหม่ → ตั้งชื่อ เช่น "Equipment Borrowing Log"
คัดลอก Spreadsheet ID จาก URL (ส่วนระหว่าง /d/ กับ /edit)
2
เปิด Apps Script
ใน Google Sheet → เมนู Extensions → Apps Script
ลบโค้ดเดิมออกทั้งหมด แล้ววางโค้ดจากไฟล์ Code.gs แทน
แก้ไข SPREADSHEET_ID ให้ตรงกับ Sheet ของคุณ
3
Deploy เป็น Web App
ใน Apps Script → กดปุ่ม Deploy → New deployment
ตั้งค่าดังนี้:
  • Type: Web app
  • Execute as: Me
  • Who has access: Anyone
กด Deploy → อนุญาต permission → คัดลอก URL
4
วาง URL ในแท็บนี้
นำ URL ที่ได้มาวางในช่องด้านบน ระบบจะเชื่อมต่อ Google Sheets อัตโนมัติ
5
ตรวจสอบความพร้อมใช้งาน
กลับไปที่แท็บ สร้างรายการ → กดปุ่ม 🔍 ตรวจสอบความพร้อมใช้
ระบบจะแสดง 🟢 พร้อมใช้ / 🔴 ถูกยืม ข้างชื่ออุปกรณ์แต่ละรายการ
Hover ที่ป้าย 🔴 เพื่อดูว่าใครยืมไปและจะคืนเมื่อไหร่
6
คืนอุปกรณ์
ไปที่แท็บ ↩️ คืนอุปกรณ์ → กด โหลดรายการ เพื่อดูอุปกรณ์ที่ถูกยืมอยู่
ติ๊กเลือกอุปกรณ์ที่นำมาคืน → กรอกชื่อผู้คืน → กด ยืนยันการคืน
ระบบจะอัปเดตสถานะอุปกรณ์เป็น "พร้อมใช้" ทันที และบันทึกลงใน Sheet รายการคืนอุปกรณ์

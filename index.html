🔗 Google Apps Script URL
วาง URL ของ Apps Script ที่นี่
ยังไม่ได้ตั้งค่า
📖 วิธีตั้งค่า Google Apps Script (ทำครั้งเดียว)
1
สร้าง Google Sheet
ไปที่ sheets.google.com → สร้าง Spreadsheet ใหม่ → ตั้งชื่อ เช่น "Equipment Borrowing Log"
คัดลอก Spreadsheet ID จาก URL (ส่วนระหว่าง /d/ กับ /edit)
2
เปิด Apps Script
ใน Google Sheet → เมนู Extensions → Apps Script
ลบโค้ดเดิมออกทั้งหมด แล้ววางโค้ดจากไฟล์ Code.gs แทน
แก้ไข SPREADSHEET_ID ให้ตรงกับ Sheet ของคุณ
3
Deploy เป็น Web App
ใน Apps Script → กดปุ่ม Deploy → New deployment
ตั้งค่าดังนี้:
  • Type: Web app
  • Execute as: Me
  • Who has access: Anyone
กด Deploy → อนุญาต permission → คัดลอก URL
4
วาง URL ในแท็บนี้
นำ URL ที่ได้มาวางในช่องด้านบน ระบบจะเชื่อมต่อ Google Sheets อัตโนมัติ
5
Deploy บน GitHub Pages
อัพโหลดไฟล์ index.html นี้ขึ้น GitHub repository
ไปที่ Settings → Pages → Source: main branch
เว็บจะพร้อมใช้ที่ https://username.github.io/repo-name
